# V19 Release Notes — Enterprise Spatial Production Layer

V19 continues the V18 Gaussian Splatting / digital twin execution layer and adds the production controls needed for a serious pilot.

## Added

- Spatial Production Orchestrator
- Dataset versioning and asset lineage map
- Human QA / scene review queue
- Tenant-safe 3D viewer deployment package
- Spatial SLA probes and monitoring targets
- Time-limited 3D share packages for seller/buyer/review audiences
- V19 readiness center and release check scripts

## Positioning

This version is designed to show that the platform can operate like a true digital-twin production system, not only a listing page builder. It still needs a live 3D reconstruction provider or GPU worker to generate real `.ksplat` / `.splat` / `.ply` artifacts.

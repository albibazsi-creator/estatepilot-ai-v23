# Spatial Acceptance Criteria

A 3D scene should not be presented as a production digital twin until it passes these gates:

- geometry score >= 72
- texture score >= 72
- coverage score >= 70
- viewer score >= 80
- checksum exists
- original media retained
- AI/3D disclosure visible
- no hidden defects or invented rooms/views
- human review completed before public publish

MVP may show the scene as an AI preview if clearly labeled.

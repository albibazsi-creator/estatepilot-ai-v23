# V9 Enterprise Handoff

This version is positioned as an agency SaaS foundation.

## Enterprise-facing pillars

1. Brand control: agencies can own the public experience.
2. Localization: listings can be packaged for HU/EN/DE buyers.
3. Buyer intelligence: personas and match scores help agents prioritize.
4. Safe AI: chat guardrails prevent unsupported claims.
5. Partner API: external portals and franchise tools can consume listing data.
6. SLA/support: operational readiness is visible.

## Demo path

1. Open dashboard.
2. Show a public listing.
3. Show white-label brand profile.
4. Generate EN/DE translation from translations page.
5. Show hot lead + deal pipeline.
6. Show buyer persona match.
7. Show guardrail event and explain hallucination prevention.
8. Show seller report and partner API contract.

## Production notes

Use this as a codebase handoff, not a fully hosted production deployment. Run the local checks, review Prisma schema changes, then wire real providers.

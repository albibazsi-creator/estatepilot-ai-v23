# V10 Release Notes — Enterprise Readiness Layer

V10 turns the previous agency SaaS demo into a more serious enterprise handoff package.

## Added

- AI Decision Ledger for auditable AI outputs.
- GDPR / DSR request center for export/delete/rectify workflows.
- Customer Success Health scoring for agencies.
- Product Feedback / NPS loop.
- Listing Improvement Engine with conversion recommendations.
- AI Evaluation Suite for property-chat guardrails.
- Backup Snapshot Center for metadata restore-readiness.
- Release Channel + customer-visible changelog.
- TypeScript SDK example and SDK contract check.
- V10 readiness API and scripts.

## Important limitation

This remains a developer handoff package. It still needs a real local `npm install`, Prisma generate, `npm run build`, provider credentials, auth hardening and deployment verification before it is sold as a live SaaS.

# EstatePilot AI v8 Demo Package

## 12 perces demo flow

1. Probléma: az ingatlanos nem látja, ki a komoly érdeklődő.
2. Listing létrehozás: ingatlanadatok, média, 360/Matterport embed.
3. Public landing oldal: prémium hirdetési élmény és lead űrlap.
4. AI tartalom: description, social, Reels, campaign package.
5. Lead scoring: 0–100 pontszám és next best action.
6. Deal pipeline: leadből forecastolt jutalékérték.
7. Proposal draft: hívási script + email follow-up.
8. Seller report: tulajdonosnak mérhető munka.
9. Data quality + compliance: publikálási kapu.
10. Árazás: bevezető listing csomag és havi Pro.

## Demo előtt ellenőrizd

```bash
npm install
npm run db:push
npm run db:seed
npm run release:v8-check
npm run dev
```

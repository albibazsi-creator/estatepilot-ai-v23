# Usage Metering

V13 adds a lightweight usage ledger for unit economics.

Tracked examples:

- `listing_hosted`
- `lead_capture`
- `seller_report`
- `ai_listing_copy`
- `ai_job`

The first purpose is not billing accuracy. The first purpose is pilot pricing validation: how much value and estimated cost each listing generates.

Dashboard: `/dashboard/metering`.
API: `/api/metering/usage`.

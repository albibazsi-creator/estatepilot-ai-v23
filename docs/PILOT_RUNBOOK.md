# Pilot Runbook

## Goal

Run the first paid or semi-paid agency pilot with one real listing and prove that EstatePilot AI creates measurable value: better listing presentation, more serious lead handling, owner-visible reporting and faster follow-up.

## Suggested pilot package

- 1 listing
- AI listing copy
- Gallery + 360/Matterport embed if available
- Public landing page
- Lead form with GDPR consent
- Lead scoring
- Seller report
- Follow-up task list
- Pilot success report after 7 days

## Day 0 — Setup

1. Create agency account.
2. Assign pilot owner.
3. Add brand profile and domain plan.
4. Confirm legal text and GDPR wording.
5. Run `npm run release:v13-check`.

## Day 1 — Listing intake

1. Collect core property facts.
2. Upload photos and floor plan.
3. Add seller email for report sharing.
4. Add Matterport/360 iframe if the agent has one.
5. Run quality and publish gate checks.

## Day 2 — Marketing package

1. Generate listing description.
2. Generate Facebook/Instagram/Reels copy.
3. Review AI staging disclosures.
4. Export campaign package.

## Day 3–7 — Lead operations

1. Monitor lead capture.
2. Use lead score to decide call priority.
3. Create follow-up tasks.
4. Send seller report.
5. Collect feedback/NPS.

## Pilot success metrics

- Landing page views
- Gallery opens
- Tour opens
- Floor plan opens
- Leads captured
- Hot leads
- Appointments booked
- Seller report opened/approved
- Agent feedback

## Close

Turn the pilot into one of these offers:

- Pay-per-listing Starter
- Pay-per-listing Pro
- Monthly Solo Agent
- Monthly Pro Agent

# V13 API Contract

The V13 contract is generated by `lib/api-contract.ts` and exposed at:

- `GET /api/contracts/openapi` — returns current OpenAPI-like contract.
- `POST /api/contracts/openapi` — saves a checksum snapshot.

The contract has two parts:

1. Manually described stable endpoints.
2. Automatically discovered `app/api/**/route.ts` endpoints.

For partner integrations, only the manually described stable endpoints should be considered supported until the endpoint is explicitly promoted from `beta` or `mock`.

## Recommended contract workflow

```bash
npm run contract:generate
npm run release:gates
npm run v13:readiness
```

Before a production release, save a snapshot and compare checksum. A changed checksum means the API surface changed and should be reviewed.

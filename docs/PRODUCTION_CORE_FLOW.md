# Production Core Flow

The core flow is the path that must work before selling this to the first real estate agent.

1. Create a listing
2. Add at least 6 media items and a cover image
3. Generate AI listing copy/social/seller summary
4. Publish the public listing landing page
5. Submit a GDPR-compliant lead
6. Score the lead
7. Generate a seller report
8. Create campaign and deal follow-up
9. Verify live provider adapters

Run:

```bash
npm run core-flow:check
```

Dashboard:

```text
/dashboard/core-flow
```

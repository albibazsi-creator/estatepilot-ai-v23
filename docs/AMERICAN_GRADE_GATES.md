# American-grade Gates

The system should not start a paid external pilot until these gates are passed or explicitly waived in a pilot contract.

1. Production build proof: install, db push, seed, typecheck, build, and manual smoke.
2. Live AI SLA: OpenAI vision/text/chat live with evals, trace, and guardrails.
3. 3D provider acceptance: scene manifest, .ksplat/.splat/.ply output, QA gate, and viewer deployment.
4. Premium UX benchmark: 2-minute Zillow/Matterport-style wow flow on one listing.
5. CRM revenue automation QA: lead-to-score-to-task-to-deal-to-report loop.
6. Provider certification: auth, storage, email, monitoring live; billing/calendar allowed as documented pilot fallback.

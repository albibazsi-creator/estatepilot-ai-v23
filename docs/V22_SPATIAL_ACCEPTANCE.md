# V22 Spatial Acceptance

A 3D scene can only be shown as pilot-ready when it has:

- scene.manifest.json
- checksum
- preview image
- .ksplat, .splat, or .ply asset
- quality.json
- coverage score >= 78
- geometry score >= 72
- texture score >= 74
- viewer score >= 85
- disclosure text
- human QA approval
